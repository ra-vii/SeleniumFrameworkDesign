# Selenium Framework Design

## 📌 Overview
This project is a Hybrid Selenium Automation Framework built using Java, TestNG, Maven, and Page Object Model (POM).  
It is designed to create scalable, maintainable, and reusable test automation scripts.

---

## 🛠 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Extent Reports (if used)
- Git (Version Control)
